# 🚢 Titanic Survival Prediction

This project builds a machine learning pipeline to predict passenger survival on the Titanic dataset using classification models and data preprocessing techniques.

---

## 📌 Overview

The goal of this project is to analyze the Titanic dataset and build predictive models that determine whether a passenger survived based on features like age, gender, class, and fare.

---

## 🚀 Features

- Data loading using Seaborn dataset
- Data cleaning and preprocessing
- Handling missing values
- Feature encoding and scaling
- Machine learning models:
  - Logistic Regression
  - Decision Tree Classifier
- Model evaluation using cross-validation
